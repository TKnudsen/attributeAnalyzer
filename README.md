# attributeAnalyzer
little visual helpers to estimate the type of raw attributes of tabular/multivariate dat

## Usage

Maven coordinates:

```xml
<dependency>
  <groupId>com.github.tknudsen</groupId>
	<artifactId>attribute-analyzer</artifactId>
	<version>0.0.1-SNAPSHOT</version>
</dependency>
```
